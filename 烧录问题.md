# 找不到设备的排查：

1、确认DriverAssitant驱动是否安装成功
2、是按住（或短接）maskrom(RECOVERY)后上电，确认接的是otg口。
3、若正常上电绿灯闪烁，按住maskrom(RECOVERY)上电后绿灯不闪烁则板卡已经进入maskrom模式，
4、可以更换usb线或者usb口尝试,有些usb口供电不足也会导致识别不了。
4、看下是否开了虚拟机，开了虚拟机会让你选择连接到哪里。
5、win11系统可能有bug,某些用户使用win11时识别不到。

先按照这几个原因排查一下

# 进入烧录模式：

1.准备一根Type-C线，一端连接电脑，一端预留备用
2.将所有有可能给板卡供电的接线都断开，如电源线，USB线等
3.按住REC按键(RECOVERY)或者是maskrom，先将预留的USB线插入板卡的USB-OTG接口，再插入电源
4.等待软件提示 发现一个LOADER设备或者是maskrom，即可松开按键
5.如果不成功，重复2-3步骤。

# lubancat-Q1进入烧录模式：

1.准备一根Type-C线，一端连接电脑，一端预留备用
2.将所有有可能给板卡供电的接线都断开，如电源线，USB线等

4.等待软件提示 发现一个MASKROM设备，再松开按键
5.如果不成功，重复2-4步骤。
