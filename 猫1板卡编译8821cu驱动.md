## 0.编译部分内核文件
因为鲁班猫的内核用的是x86_64架构的服务器编译的，所以要自己编译部分文件
```
cd /usr/src/linux-headers-`uname -r`/scripts/basic
sudo rm fixdep
sudo gcc fixdep.c -o fixdep
cd /usr/src/linux-headers-`uname -r`/scripts/mod/
sudo rm modpost
sudo gcc file2alias.c sumversion.c modpost.c -o modpost
cd /usr/src/linux-headers-`uname -r`/scripts
sudo rm recordmcount
sudo gcc recordmcount.c -o recordmcount
```
